# Análisis de Frecuencia de Números — Loterías Animalitos

Este proyecto analiza la frecuencia de aparición de números ganadores en las loterías **GRANJA**, **SELVA** y **LOTTO_ACTIVO** durante los últimos 3 días.

## 📊 Descripción
El notebook `frecuencia_ultimos_3_dias.html` consulta una base SQLite y genera una visualización interactiva con Plotly.
